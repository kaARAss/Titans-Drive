# Титаны Драйва — одностраничный сайт (HTML + GSAP)

Сайт на базе открытого шаблона Codrops **«Consecutive Scroll / One Element»** (MIT),
перекроен под Титаны Драйва: одно фото байка «путешествует» по секциям на скролле
(GSAP Flip + ScrollTrigger + Lenis).

## Как открыть

Это обычный HTML — **не нужны Node/npm**. Но из-за ES-модулей нужен локальный сервер
(просто двойной клик по index.html не заработает — браузер блокирует модули с file://).

Самый простой способ — одна команда в папке проекта:

```bash
# если есть Python (обычно есть):
python -m http.server 5500
```
Затем открой http://localhost:5500

Или через VS Code — расширение **Live Server** → правая кнопка по index.html → «Open with Live Server».

⚠ Нужен интернет при первом открытии — GSAP, Lenis и шрифты грузятся с CDN.

## Где что менять

- **Весь текст, туры, цены, контакты** — прямо в `index.html`.
- **Фото** — папка `img/` (photo-1..23.webp). Меняй файлы или ссылки `url(img/...)` в index.html.
- **Цвета/шрифты** — верх `css/base.css` (переменные --color-*, --gradient-*).
- **Фото, которое “путешествует”** — `.one` в index.html (сейчас photo-4).
